# GardenPal Capstone Project
This is a website created for Rutgers ECE Capstone project called GardenPal, meant to 
track plant data, weather changes, and predator appearences in the user's garden or 
backyard.

## Login Page
This page allows the user to log into their account. 

## Plant Data Page
This page shows the plants within the user's garden as well as the indivisual plants' 
sunlight level, humidity levels, and soil moisture levels

## Weather Page
This page shows the weather in the user's area, the precipitation graph, which is useful
information for soil moisture levels, and a weekly weather forecast. Connected to a weather
API to display accurate weather data.

## Dashboard
This page shows the garden camera feed, as well as any alerts that may have came in that day,
such as sunlight levels or predator detections.

## Account Page
This page allows the user to view and change their email, password, and location easily
